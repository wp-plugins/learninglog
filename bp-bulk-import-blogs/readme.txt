BuddyPress Bulk Import Blogs ermöglicht den Massenimport von Weblogs.

== Description ==

#### Was ist BuddyPress Social Access?

BuddyPress Social Access ermöglicht den Zugang zu Blogposts nur für bestimmte Nutzer oder Gruppen zuzulassen. 

## Superadmin kann:
* Zugang zu Massenimport gewähren 
* Zugang zu Massenimport ablehnen

## Nutzer kann:
* Zugang zu Massenimport beantragen
* Wenn Zugang gewührt wurde: in einem Textfeld mehrere Nutzer erfassen und importieren
* Nutzer beim Import einer Gruppe zuordnen
* Neue Nutzer erhalten eine E-Mail mit den entsprechenden Login-Daten zugesendet.

== Installation ==

Simplicity requires WordPress 3.0

####Plugin:

1. Upload everything into the "/wp-content/plugins" directory of your installation.
2. Activate BuddyPress Links in the "Plugins" admin panel using the "Activate" link.
