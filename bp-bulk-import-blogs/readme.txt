BuddyPress Bulk Import Blogs ermoeglicht den Massenimport von Weblogs.

## Superadmin kann:
* Zugang zu Massenimport gewaehren 
* Zugang zu Massenimport ablehnen

## Nutzer kann:
* Zugang zu Massenimport beantragen
* Wenn Zugang gewaehrt wurde: in einem Textfeld mehrere Nutzer erfassen und importieren
* Nutzer beim Import einer Gruppe zuordnen
* Neue Nutzer erhalten eine E-Mail mit den entsprechenden Login-Daten zugesendet.
