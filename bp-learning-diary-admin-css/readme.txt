== Description ==

#### Was istBuddyPress Learning Diary Admin Template?

BuddyPress Learning Diary Admin Template verändert die Admin-Umgebung so, dass Backend und Frontend sehr ähnlich sind:
A: verschiebt die Admin-Navigation von der rechten auf die linke Seite
B: fügt den Buddypress Header im Kopfbereich ein
C: Versteckt den Administrations-Header

== Installation ==

Simplicity requires WordPress 3.0

####Plugin:

1. Upload everything into the "/wp-content/plugins" directory of your installation.
2. Activate BuddyPress Links in the "Plugins" admin panel using the "Activate" link.
