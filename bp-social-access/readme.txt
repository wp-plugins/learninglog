== Description ==

#### Was ist BuddyPress Social Access?

BuddyPress Social Access ermöglicht den Zugang zu Blogposts nur für bestimmte Nutzer oder Gruppen zuzulassen. 

## Nutzer können aufgrund ihrer Gruppenzugehörigkeit:
* einer oder mehreren Gruppen Zugang zu einem Post gewähren
* einzelnen Nutzern Zugang zu einem Post gewähren
* bestimmen, ob RSS-Feeds die geschützten Beiträge ausblenden oder den Titel (ohne Inhalt) anzeigen

== Installation ==

Simplicity requires WordPress 3.0

####Plugin:

1. Upload everything into the "/wp-content/plugins" directory of your installation.
2. Activate BuddyPress Links in the "Plugins" admin panel using the "Activate" link.
