== Description ==

#### Was ist Simplicity?

Simplifies Wordpress (Removes BuddyPress Admin Bar and Dashboard Items as well as some Menus and Submenus)

This is a one-button one-click Plugin to enable or disable the simple mode.

== Installation ==

Simplicity requires WordPress 3.0

####Plugin:

1. Upload everything into the "/wp-content/plugins" directory of your installation.
2. Activate Simplicity in the "Plugins" admin panel using the "Activate" link.
